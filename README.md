# torchdrug

## Installation

```bash
conda create -ndrug python=3.8
conda activate drug
conda install torchdrug -c milagraph -c conda-forge -c pytorch -c pyg
pip install -r requirements.txt -i"https://pypi.tuna.tsinghua.edu.cn/simple/"
```
